COMANDOS GIT

  • Para checar a versão do git: git --version
  • Clonar um repositório: git clone <'url do repositorio'>
  • Adicionar todos os itens modificados na lista para o commit: git add .
  • Para adicionar a fila de itens que serão mandados para o repositorio: git commit -m "nome do arquivo"
  • Para enviar todos os itens commitados para o repositorio: git push
